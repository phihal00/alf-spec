# Changelog

## [1.0.0-draft] — 2026-06-25

Initial public draft of the ALF v1 specification.

- Nine-section structure defined
- Four layer types: reasoning, procedure, context, data
- Four display types: expert, editorial, institutional, community
- Conformance criteria for files and readers
- JSON Schema at spec/schema-v1.json
- Two reference examples: minimal reference, clinical differential reasoning benchmark
- Open/private boundary statement (§14): structural protection only, no cryptographic claims
- llms.txt for LLM agent indexing
