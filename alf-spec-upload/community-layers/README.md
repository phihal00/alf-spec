# Community Layers

This directory contains ALF layers contributed by the open community under permissive licences.

## Contributing a community layer

See [CONTRIBUTING.md](../CONTRIBUTING.md) for the full process.

Each file here carries its own licence in `metadata.licence`. Community layers use `"display_type": "community"` and `"licence": "CC BY 4.0"` unless otherwise specified.

## Quality bar

Community layers are reviewed against the ALF v1 conformance criteria before acceptance. Structural conformance (schema validation) is a gate; content quality (unknown unknowns, enhancement template bar) is reviewed by a maintainer.
